uglify
=======================================================================================================================

uglify用于压缩代码的。 

官网
-----------------------------------------------------------------------------------------------------------------------
http://lisperator.net/uglifyjs/


安装 uglify
-----------------------------------------------------------------------------------------------------------------------

### 全局

```shell
npm i uglify-js -gd
```

### 项目
```shell
npm i uglify-js --save-dev -d
```

编译客户端
-----------------------------------------------------------------------------------------------------------------------
```shell
uglifyjs --self -c -m -o /tmp/uglifyjs.js
```
